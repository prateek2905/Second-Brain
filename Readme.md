# Second Brain

A simple app to capture ideas, thoughts, useful links, and organize everything with tags.

## Why This Project

Keeping notes in scattered places makes it hard to find things later.  
Second Brain is a lightweight personal knowledge space where you can quickly save and rediscover what matters.

## Features (MVP)

- Save quick ideas and thoughts
- Store useful links with notes
- Add tags to organize entries
- Browse and search saved content
- Keep everything in one place

## Example Use Cases

- Save startup ideas as they come to mind
- Keep articles and videos for later reading
- Group notes by tags like `productivity`, `learning`, `business`, `health`
- Track random thoughts you want to revisit

## Tech Stack

- Node.js
- TypeScript

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the app:

```bash
npm start
```
